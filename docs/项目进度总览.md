## 六、关键风险

1. **SaToken 鉴权被注释** — 当前所有接口无认证保护，上线前必须恢复
2. **API 前缀 `/api` vs `/api/v1`** — 前端可能按 `/api/v1/` 调用导致 404
3. **AI 问答为 stub** — BE3 需尽快接入 MaxKB，否则智能体功能无法演示
4. **前端只有构建产物无源码** — 如前端需修改，需联系 FE1/FE2 获取 Vue 源码
